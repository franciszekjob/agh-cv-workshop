# AI Workshop: CV, CNN, YOLO

Notebook prepared for a workshop for high school students at AGH University of Krakow, held on 16.05.2026.

## Contents

- Computer Vision — what it is, how a computer sees an image, pixel-level math (negative, brightness)
- CV problem types — classification, localization, object detection, instance segmentation
- Neural networks in a nutshell — neurons, activation functions (ReLU), layers, training
- CNNs — convolutions, Sobel filters, feature hierarchy, why CNNs beat plain MLPs on images
- MNIST demo — training a small CNN to recognize handwritten digits
- YOLO — from classification to real-time object detection
